# Product Context

## Problem Statement

Emergency response organizations need efficient coordination between field responders and supervisors during incidents. Traditional communication methods (radio, phone) lack:
- Real-time location visibility
- Document capture and signing workflows
- Incident history and audit trails
- Efficient route planning for multiple incidents

## Solution

NFA Alerts provides a unified platform where:
- **Supes** dispatch and monitor Chasers in real-time
- **Chasers** receive incident alerts, respond, and document their work
- **Everyone** communicates through secure, role-based chat
- **Documents** are captured, signed, and stored digitally

## User Journeys

### Chaser Flow
1. Receive push notification for new incident
2. View incident details (address, type, alarm level)
3. Respond to incident (adds to their active incidents)
4. Navigate using route planner
5. On-site: capture photos, collect homeowner info
6. Get homeowner signature + sign themselves
7. Generate signed document PDF

### Supe Flow
1. Monitor all active incidents and chaser locations
2. View real-time chaser status (Responding/Available/Offline)
3. Communicate with chasers via chat
4. Award "Secured" status when chaser wins homeowner
5. Remove responders or close incidents as needed

### Admin Flow
1. Manage user accounts (approve, suspend, ban)
2. Configure locations/regions
3. View system-wide activity

## Key Differentiators
- **Mobile-first PWA** - Works like native app without app store
- **Real-time everything** - Incidents, chat, presence updates instantly
- **Digital document workflow** - No paper, immediate PDF generation
- **Role-based intelligence** - UI adapts to user role
- **Interactive onboarding** - Guided walkthrough for new users
